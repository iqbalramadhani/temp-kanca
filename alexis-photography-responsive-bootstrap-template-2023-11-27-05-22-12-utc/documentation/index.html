Fonts Source File structure

Remember to change the font into main.css

            
            <!-- Fonts -->
            @import url(https://fonts.googleapis.com/css?family=DM+Sans:400,400i,500,700,|Libre+Baskerville:400,400i,700);
            @import url('https://fonts.googleapis.com/css2?family=Poppins&display=swap');


CSS Source File structure

These are the css files that are loaded into templates in Head Section.

            
            <!-- Main Style CSS -->
            <link href="assets/css/style.css" rel="stylesheet">


Javascript Source File structure

These are the list of Javascript files that are loaded into templates in end of the Body Section.

    
            <!--=======================Javascript============================-->

<!--=== Modernizr Min Js ===-->
<script src="assets/js/modernizr.js"></script>

<!--=== jQuery Migration Min Js ===-->

<script src="assets/js/bootstrap.min.js"></script>
<!--=== jquery UI Min Js ===-->
